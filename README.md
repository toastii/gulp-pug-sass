# Gulp Pug Sass

This is a simple starter project using **Pug**, **Sass/Scss** and **browser-sync** with for **gulp** for task automation.

if you don't have **node.js** you can get it [here](https://nodejs.org).


## Getting started

Execute:

- npm install gulp

- npm install

- gulp watch

## Packages

- gulp
- gulp-pug
- gulp-sass
- browser-sync
- gulp-autoprefixer
- del
- gulp-concat
- gulp-plumber
